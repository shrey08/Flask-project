INSTRUCTIONS:

We need to install virtual environment to run application.
Inside the Flask-project folder, we need to install virtual environment.
(First install python)

Commands:
py -3 -m venv <give any name to folder>
Now we'll seea folder made with many files inside it.

To activate virtual environment:
<folder name>\Scripts\activate
This will acivate the virtual environment.

To install flask:
pip install flask   

Inside static folder install bootstrap.
Some files will be created named as: bootstrap.min.css, bootstrap.min.js, jquery-3.3.1.slim.min.js, popper.min.js
